## PDFProcessor工具使用

```python
from OmniText.PDFProcessor import *

# 初始化处理器
processor = PDFProcessor(output_dir="output")

# 处理多个文件
files = [
    r"文献综述1.pdf",
    r"文献综述2.pdf",
    r"文献综述4.pdf"
]
processor.process(files)

# 获取合并结果字符串
combined_result = processor.get_output(combine=True)
print(f"合并结果长度: {len(combined_result)} 字符")

# 保存为合并文件
processor.save_as_txt(combine=True, output_path="output/全部结果.txt")

# 保存为单独文件
processor.save_as_txt(combine=False)
```
